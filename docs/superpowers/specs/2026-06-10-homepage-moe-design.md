# ANISpace 首页改版设计

## 设计方向
- **风格**：二次元萌系 × 图片驱动
- **布局**：方案 A — 大封面横滑推荐 + 7:3 两栏
- **配色**：基于现有 `--primary: #e886a2` 粉色系 + `--secondary: #7eb8da` 天蓝辅助色
- **核心元素**：大封面图、圆角卡片、柔和阴影、标签药丸、emoji 点缀

## 首页结构

### 1. 顶部横滑推荐区（Hero Carousel）
- 高度约 200px，圆角 16px
- 每张卡片：左侧渐变背景 + 右侧半透明封面缩略图
- 卡片内容：推荐标签（🌸 今日推荐）、标题、评分+类型、标签药丸、操作按钮
- 底部横滑指示器（圆点）
- 最多展示 5 张，可左右横滑

### 2. 两栏布局（7:3）

#### 左栏（70%）
**每日放送**
- 星期切换标签（药丸样式，当前选中为 primary 渐变）
- 5 列网格，3:4 封面比例
- 每个封面右上角评分角标（白底圆角）
- 标题 + 放送时间

**热门讨论**
- 2 列卡片网格
- 每张卡片：头像 + 用户名 + 时间 + 内容 + 互动数据
- 圆角 14px，柔和阴影

#### 右栏（30%，sticky）
**行业资讯**
- 左侧彩色边框条（primary/secondary 交替）
- 标题 + 时间

**世界频道预览**
- 白色圆角卡片
- 头像 + 用户名 + 时间 + 内容
- 底部"进入频道 →"链接

## CSS 变量使用
- 所有颜色使用现有 CSS 变量（`--primary`, `--secondary`, `--bg-card` 等）
- 圆角：`--radius-lg` (16px) 用于大卡片，`--radius-md` (12px) 用于小卡片
- 阴影：`--shadow-sm` 用于小卡片，`--shadow-md` 用于大卡片

## 响应式
- 1024px 以下：单栏布局
- 768px 以下：每日放送 3 列，热门帖子 1 列
- 480px 以下：每日放送 2 列，横滑推荐高度缩小
